# 人物弧光

本篇有三个主要人物

情感文的人物弧光体现在

------

”人物弧光“
人物在故事开始是持有某一观点，在突破重要障碍实现目标的过程中，经历考验和磨难，致使这一观点、人物的状态和品质发生了改变。

实质上，人物弧光是人物情感和心理的成长、转变和发展。

------
